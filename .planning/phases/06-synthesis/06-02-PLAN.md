    1. **Defensive empty-selection guard:** if `selection.winner_cluster_id is None and selection.fallback_article_id is None`: raise `ValueError("run_synthesis called with empty selection — caller must short-circuit")`.

    2. **Determine input articles + cluster metadata:**
       - Winner branch (`winner_cluster_id is not None`):

         - `centroid_terms = cluster_row.centroid_terms or {}`
       - Fallback branch:
         - `article = session.get(Article, selection.fallback_article_id)`; if None → raise ValueError
         - `selected = [article]`
         - `source_url = article.url`
         - `cluster_id = None`
         - `theme_centroid = None`
         - `centroid_terms = {}`

    3. **Hashtags (D-05, D-11):**
       - `hashtags = select_hashtags(centroid_terms, hashtag_allowlist, top_k=10, max_tags=2)`
       - Enforce weighted budget: while `weighted_len(" ".join(hashtags)) > settings.hashtag_budget_chars and len(hashtags) > 1`: drop last tag. (Guards against pathologically long tags bursting the 30-char reservation.)

    4. **Body budget:** `body_budget = settings.synthesis_char_budget` (default 225).

    5. **Retry loop (D-06):**
       ```python
       system = build_system_prompt(body_budget)
       user_prompt = build_user_prompt(selected)
       attempt_log: list[dict] = []
       total_input_tokens = 0
       total_output_tokens = 0
       body_text: str | None = None
       final_method: Literal["completed", "truncated"] = "completed"
       max_attempts = settings.synthesis_max_retries + 1   # 1 initial + N retries
       current_prompt = user_prompt

       for attempt in range(1, max_attempts + 1):
           text, in_tok, out_tok = call_haiku(
               anthropic_client, system, current_prompt, settings.synthesis_max_tokens,
           )
           total_input_tokens += in_tok
           total_output_tokens += out_tok
           candidate_final = format_final_post(text, source_url, hashtags)
           cand_len = weighted_len(candidate_final)
           attempt_log.append({"attempt": attempt, "length": cand_len, "text_preview": text[:120]})
           log.info("synth_attempt", attempt=attempt, length=cand_len, input_tokens=in_tok, output_tokens=out_tok)
           if cand_len <= 280:
               body_text = text
               break
           # prep retry prompt for next iteration
           current_prompt = build_retry_prompt(text, cand_len, body_budget - 10)
       else:
           # loop exhausted without break → last-resort truncation
           # Compute remaining budget for body after URL (23) + 1 space + hashtags
           hashtag_block = " ".join(hashtags)
           overhead = 23 + 1 + (weighted_len(hashtag_block) + 1 if hashtag_block else 0)
           body_budget_hard = 280 - overhead
           body_text = word_boundary_truncate(text, body_budget_hard)  # text = last attempt
           final_method = "truncated"
           log.warning("synth_truncated", final_length_estimate=weighted_len(format_final_post(body_text, source_url, hashtags)))

       assert body_text is not None  # either break or truncate path sets it
       final_text = format_final_post(body_text, source_url, hashtags)
       assert weighted_len(final_text) <= 280, "invariant violated"
       attempts_count = len(attempt_log)
       ```

    6. **Cost:** `cost_usd = compute_cost_usd(total_input_tokens, total_output_tokens)`.

    7. **Status (D-12):** `status = "dry_run" if settings.dry_run else "pending"`.

    8. **error_detail (D-10):** `error_detail = json.dumps(attempt_log) if final_method == "truncated" else None`.

    9. **Persist (D-08/09/10):**
       ```python
       post = insert_post(
           session,
           cycle_id=cycle_id,
           cluster_id=cluster_id,
           status=status,
           theme_centroid=theme_centroid,
           synthesized_text=final_text,
           hashtags=hashtags,
           cost_usd=cost_usd,
           error_detail=error_detail,
       )
       session.flush()
       ```

    10. **Log + return:**
        ```python
        log.info(
            "synth_done",
            attempts=attempts_count,
            final_method=final_method,
            input_tokens=total_input_tokens,
            output_tokens=total_output_tokens,
            cost_usd=cost_usd,
            post_id=post.id,
        )
        return SynthesisResult(
            text=final_text,
            body_text=body_text,
            hashtags=hashtags,
            source_url=source_url,
            attempts=attempts_count,
            final_method=final_method,
            input_tokens=total_input_tokens,
            output_tokens=total_output_tokens,
            cost_usd=cost_usd,
            post_id=post.id,
            status=status,
            counts_patch={
                "synth_attempts": attempts_count,
                "synth_truncated": final_method == "truncated",
                "synth_input_tokens": total_input_tokens,
                "synth_output_tokens": total_output_tokens,
                "synth_cost_usd": cost_usd,
                "post_id": post.id,
            },
        )
        ```
